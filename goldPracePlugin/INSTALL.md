# 安装指南

## 安装步骤

```
1. 打开 chrome://extensions/
2. 开启"开发者模式"
3. 点击"加载已解压的扩展程序"
4. 选择本项目文件夹
5. 完成
```

## 使用

- 安装后自动运行，每秒更新金价
- 徽章显示当前价格
- 点击图标查看趋势图
- 绿色价格 = 实时数据，橙色 = 模拟数据

## 验证

打开 `chrome://extensions/` → 点击 "Service Worker" 查看日志：

```
✅ 成功获取金价: 917.39 元/克
```

## 故障排查

**显示模拟数据**：检查网络，确保可访问 `papi.icbc.com.cn`  
**价格不更新**：重新加载扩展  
**Service Worker 无输出**：点击 "Service Worker" 激活
