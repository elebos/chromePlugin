# 实时金价 Chrome 扩展

实时显示上海黄金交易所金价的轻量级 Chrome 扩展。

## 功能

- 工具栏实时显示 Au99.99 金价
- 点击查看全天价格走势图
- 每秒自动更新
- 显示交易状态（交易中/已休市）
- 后台静默运行

## 安装

```
chrome://extensions/ → 开发者模式 → 加载已解压的扩展程序
```

## 数据源

**API**: `https://www.sge.com.cn/graph/quotations`  
**来源**: 上海黄金交易所官方API  
**产品**: Au99.99（黄金现货）  
**单位**: 元/克  
**更新**: 每秒  

**交易时间**：
- 周一至周五 09:00-15:30（日盘）
- 周一至周五 19:50-02:30（夜盘）
- 节假日休市

## 设计模式

- **单例模式** - GoldPriceManager 数据管理
- **观察者模式** - 价格更新通知
- **策略模式** - 数据获取和渲染

## 技术栈

- Manifest V3
- 原生 JavaScript
- Canvas API
- Chrome Storage API
